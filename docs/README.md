# Dead Code Detector Documentation

This folder contains the official documentation website for the Dead Code Detector Gradle plugin.

All pages are SEO‑optimized and include proper GEO meta tags (geo.region, geo.placename, ICBM) to target users in Iran, authored by **Aria Danesh**.

## Pages

- `index.html` – Home
- `installation.html` – Installation guide
- `configuration.html` – Full configuration reference
- `report.html` – Interactive HTML report template
- `ci.html` – CI/CD integration (GitHub Actions)
- `contributing.html` – How to contribute
- `contact.html` – Contact information
